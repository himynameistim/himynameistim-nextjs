This is an [Astro](https://astro.build/) project.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:4321](http://localhost:4321) with your browser to see the result.

## Building

To build the project for production:

```bash
npm run build
```

## Learn More

The easiest way to deploy your Astro app is to use the [Vercel Platform](https://vercel.com) or [Netlify](https://www.netlify.com/).

Check out the [Astro deployment documentation](https://docs.astro.build/en/guides/deploy/) for more details.
